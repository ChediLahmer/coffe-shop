Ce site web est dédié à notre café, qui propose une sélection de cafés de qualité que vous pouvez acheter dans la section "Menu". 
Après avoir ajouté vos articles au panier, vous pouvez finaliser votre commande en cliquant sur l'icône du panier. 
Dans la section "Blog", vous pouvez lire des articles intéressants sur le café et l'industrie du café. 
Pour réserver une table, vous pouvez vous rendre sur la page "book". 
Vous avez également la possibilité de vous connecter ou de vous inscrire sur le site.

Nous avons utilisé PHP pour le côté serveur de l'application.
Ainsi que Html, css et javascript pour le front end.
